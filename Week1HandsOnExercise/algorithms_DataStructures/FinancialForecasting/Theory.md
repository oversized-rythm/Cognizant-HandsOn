## 1. Recursion
Recursion is a technique where a method calls itself until a base condition is reached. It simplifies repetitive problems by breaking them into smaller subproblems.

## 2. How to optimize the recursive solution
The recursive solution can be optimized using the following techniques:

- **Memoization:** Stores previously computed results to avoid redundant recursive calls.
- **Dynamic Programming:** Solves the problem iteratively by building solutions from smaller subproblems, eliminating recursion overhead.
- **Iterative Approach (Loop):** Replaces recursion with a loop, reducing memory usage by avoiding the recursive call stack.

These optimizations improve performance and make the algorithm more efficient, especially for large input sizes.
