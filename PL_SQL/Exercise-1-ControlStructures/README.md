# Exercise 1 - Control Structures

## Objective
Use PL/SQL control structures to update customer and loan records.

## Scenarios
1. Apply 1% discount to loan interest rates for customers above 60.
2. Set IsVIP flag to TRUE for customers with balance over 10000.
3. Print reminder messages for loans due within the next 30 days.

## Concepts Used
- PL/SQL blocks
- FOR LOOP
- IF condition through WHERE clause
- UPDATE statement
- DBMS_OUTPUT.PUT_LINE
- COMMIT
