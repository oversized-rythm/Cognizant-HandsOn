# Week 05 - ReactJS & Git

This branch contains the mandatory hands-on exercises completed during **Week 05** of the Cognizant Digital Nurture 5.0 Java FSE program.

## Topics Covered

- React Components
- Class Components
- Functional Components
- Props
- JSX
- ES6 Features
- Event Handling
- Conditional Rendering
- React Lifecycle Methods
- Lists & Keys
- Styling Components
- Git Version Control
  - Basic commands (add, commit, push)
  - GitIgnore files
  - Branching and Merging
  - Merge Conflicts
