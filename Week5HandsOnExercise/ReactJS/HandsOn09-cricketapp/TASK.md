## Hands-on 9 : ES6 Features in React

### Objective

Create a React application that demonstrates the use of **ES6 features** such as `map()`, arrow functions, destructuring, array merging using the spread operator, and conditional rendering.

---


## Steps Performed

1. Created a new React application named **cricketapp**.
2. Created the `ListofPlayers` component.
3. Declared an array of 11 players with their names and scores.
4. Displayed all players using the `map()` method.
5. Filtered players with scores below 70 using arrow functions.
6. Created the `IndianPlayers` component.
7. Used array destructuring to display odd and even team players.
8. Merged the `T20Players` and `RanjiTrophyPlayers` arrays using the spread operator.
9. Displayed the merged list of players.
10. Used a flag variable with conditional rendering to display different components.
11. Executed the application using:

    ```bash
    npm start
    ```

12. Verified the output in the browser.
