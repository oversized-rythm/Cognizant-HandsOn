# WEEK 05 : ReactJS

<hr>

## Hands-on 1 : Create Your First React Application

### Objective

Set up a React development environment by creating and running your first React application using **Create React App**. Display a simple heading on the home page.

---

## Software Requirements

- Node.js
- npm
- Visual Studio Code
- React (Create React App)

---

## Steps Performed

1. Created a new React application named **myfirstreact**.
2. Navigated to the project directory.
3. Opened the project in Visual Studio Code.
4. Modified the `src/App.js` file.
5. Displayed the heading:

   ```text
   Welcome to the first session of React
   ```

6. Started the React development server using:

   ```bash
   npm start
   ```

7. Verified the application by opening:

   ```text
   http://localhost:3000
   ```
