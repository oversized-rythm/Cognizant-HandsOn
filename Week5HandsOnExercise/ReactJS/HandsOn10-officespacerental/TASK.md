## Hands-on 10 : Using JSX in React

### Objective

Create a React application that demonstrates the use of **JSX**, JavaScript expressions, attributes, inline CSS, and rendering multiple office space details using React components.

---

## Steps Performed

1. Created a new React application named **officespacerentalapp**.
2. Added an image to represent the office space.
3. Created a JSX element to display the page heading.
4. Defined an object containing office details such as:
   - Name
   - Rent
   - Address
   - Image
5. Created a list of office space objects.
6. Rendered the office details dynamically using the `map()` method.
7. Applied inline CSS to display:
   - **Red** color for rent values less than or equal to **₹60,000**.
   - **Green** color for rent values greater than **₹60,000**.
8. Displayed the office image, name, rent, and address using JSX.
9. Executed the application using:

   ```bash
   npm start
   ```

10. Verified the output in the browser.
