## Hands-on 3 : Creating a Functional Component

### Objective

Create a React application for a **Student Management Portal** using a **functional component**. Accept student details as props, calculate the student's score, and apply CSS styling to the component.

---

## Software Requirements

- Node.js
- npm
- Visual Studio Code
- React (Create React App)

---

## Steps Performed

1. Created a new React application named **scorecalculatorapp**.
2. Created a **Components** folder inside the `src` directory.
3. Created a functional component named `CalculateScore.js`.
4. Passed student details (`Name`, `School`, `total`, and `goal`) as props.
5. Calculated the student's score and displayed the result.
6. Created a **Stylesheets** folder and added `mystyle.css` to style the component.
7. Imported and rendered the `CalculateScore` component in `App.js`.
8. Started the React development server using:

   ```bash
   npm start
   ```

9. Verified the application by opening:

   ```text
   http://localhost:3000
   ```
