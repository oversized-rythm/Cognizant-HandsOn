## Hands-on 11 : React Event Handling

### Objective

Create a React application that demonstrates **React event handling**, including button click events, passing arguments to event handlers, synthetic events, and a simple currency converter.

---

## Steps Performed

1. Created a new React application named **eventexamplesapp**.
2. Implemented an **Increment** button to increase the counter value.
3. Implemented a **Decrement** button to decrease the counter value.
4. Invoked multiple methods using the **Increment** button to:
   - Increase the counter value.
   - Display a greeting message.
5. Created a **Say Welcome** button that passes an argument to an event handler.
6. Implemented a synthetic event using the **Click on me** button to display a message.
7. Created a **CurrencyConverter** component.
8. Accepted the amount and currency as user input.
9. Implemented the `handleSubmit()` event to perform the currency conversion.
10. Displayed the converted currency value using an alert message.
11. Executed the application using:

    ```bash
    npm start
    ```

12. Verified the application by testing all event handlers and the currency converter.
