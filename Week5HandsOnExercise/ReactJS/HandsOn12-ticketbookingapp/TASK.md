## Hands-on 12 : Conditional Rendering in React

### Objective

Create a React application that demonstrates **conditional rendering** by displaying different views for guest users and logged-in users.

---

## Steps Performed

1. Created a new React application named **ticketbookingapp**.
2. Created separate components for:
   - `LoginButton`
   - `LogoutButton`
   - `Greeting`
   - `UserGreeting`
   - `GuestGreeting`
3. Managed the login status using React state.
4. Implemented conditional rendering to display:
   - Guest page when the user is not logged in.
   - User page after successful login.
5. Displayed the **Login** button for guest users.
6. Displayed the **Logout** button for logged-in users.
7. Updated the application view dynamically when the login state changed.
8. Executed the application using:

   ```bash
   npm start
   ```

9. Verified the application by testing both the login and logout functionality.
