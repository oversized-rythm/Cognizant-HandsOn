## Hands-on 2 : Creating React Components

### Objective

Create a React application for a **Student Management Portal** using multiple **class components**. Render the components from the main application.

---

## Steps Performed

1. Created a new React application named **StudentApp**.
2. Created a **Components** folder inside the `src` directory.
3. Created the following class components:
   - `Home.js`
   - `About.js`
   - `Contact.js`
4. Implemented each component to display its respective welcome message.
5. Imported all three components into `App.js`.
6. Rendered the `Home`, `About`, and `Contact` components from the main application.
7. Started the React development server using:

   ```bash
   npm start
   ```

8. Verified the output in the browser at:

   ```text
   http://localhost:3000
   ```
