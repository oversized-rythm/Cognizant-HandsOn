## Hands-on 13 : Rendering Lists and Components in React

### Objective

Create a React application that demonstrates rendering multiple components, displaying lists using the `map()` function, assigning unique keys, and implementing conditional rendering.

---


## Steps Performed

1. Created a new React application named **bloggerapp**.
2. Created separate components for:
   - `BookDetails`
   - `BlogDetails`
   - `CourseDetails`
3. Stored the required data using JavaScript objects and arrays.
4. Rendered the list of books using the `map()` function.
5. Rendered the list of blogs using the `map()` function.
6. Rendered the list of courses using the `map()` function.
7. Assigned unique keys while rendering lists.
8. Displayed all three components on the same page.
9. Executed the application using:

   ```bash
   npm start
   ```

10. Verified the rendered output in the browser.
