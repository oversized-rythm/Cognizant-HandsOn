# Cognizant Digital Nurture 5.0 (DN5.0) Practice Assignments

This repository contains my complete set of hands-on practice assignments completed for the **Cognizant Digital Nurture 5.0 (DN5.0) Java FSE Program**. 

The exercises are structured week-by-week and organized into separate branches matching the official training program structure.
